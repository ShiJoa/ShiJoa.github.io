物理层的主要任务就是确定与传输媒体的借口有关的一些特性，如机械特性、电气特性、功能特性和过程特性。

2-5

物理层的接口特性有：
机械特性 指明接口所用接线器的形状和尺寸、引脚数目和排列、固定和锁定装置等。平时常见的各种规格的接插件都有严格的标准化和规定。
电气特性 指明在接口电缆的各条线上出现的电压的意义
功能特性 指明某条线上出现的某一电平的电压的意义
过程特性 指明对于不同功能的各种可能时间的出现顺序

1101011011/10001=1111
答：1111
如果要发送的数据在传输过程中最后一个1变成了0。接收端是否能够发现
答：发送的数据为11010110111111
接收的数据为11010110101111
11010110101111%10001 != 0
所以能发现

答：发送的数据为11010110111111
接收的数据为11010110001111
11010110001111%1111=10001
