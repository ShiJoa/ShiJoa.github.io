注册表操作

2020年12月18日 657 字 大概 3 分钟
  1. 1. 打开或创建一个注册表
  2. 2. 关闭注册表
  3. 3. 读取函数
  4. 4. 写入函数
  5. 5. 删除键值函数

在学习更高级的door木马之前。先学习一下需要用到的注册表操作吧

注册表是以树形结构存储的，树的每个节点称为一个键值（key），每一个key又包括子键值(subkey)即数据入口的值(value)。而value的具体值则包括了string、dword等。
可以在运行输入 regedit 调出注册表编辑器

打开或创建一个注册表

在读写注册表前，必须先将目标的子键打开，也就是取得一个操作的句柄。
最常用的函数就是RegCreateKeyEx，如果没有这个键存在，那么就会自动创建这个子键项。
下面是这个函数的原型
LONG RegCreateKeyEx(
HKEY hKEY,
LPCTSTR lpSubKey,
DWORD Reserved,
LPTSTR lpClass,
DWORD dwOptions,
REGSAM samDesired,
LPSECURITY_ATTRIBUTES lpSecurityAttributes,
PHKEY phkResult,
LPDWORD lpdwDisposition
);
HKEY hKey:
父键的句柄
LPCTSTR lpSubKey:
子键的名称
DWORD Reserved:
一般为0
LPTSTR lpClass:
一般为0
DWORD dwOptions:
创建子键时的选项
REGSAM samDesired:
打开方式
LPSECURITY_ATTRIBUTES lpSecurityAttributes:
指定继承性
PHKEY phkResult:
用于返回子键的句柄
LPDWORD lpdwDisposition:
打开还是创建子键的信息

如果调用成功返回ERROR_SUCCESS，即返回0，失败返回非0值

  • SamDesired 参数有多个取值，最后的lpdwDisposition 参数用于返回信息，有两个取值：
    • REG_CREATED_NEW_KEY: 指定的键不存在，那么创建
    • REG_OPENED_EXISTING_KEY: 指定的键存在，那么打开

类似的打开函数还有
LONG RegOpenKeyEx(
HKEY hKey,
LPCTSTR lpSubKey,
DWORD ulOptions,
REGSAM samDesired,
PHKEY phkResult
);
HKEY hKey:
要打开的主键句柄
LPCTSTR lpSubKey：
要打开的子键
DWORD ulOptions：
保留
REGSAM samDesired：
安全存取参数
PHKEY phkResult：
打开键句柄的地址

关闭注册表

比起打开，关闭要容易得多
LONG RegCloseKey(HKEY hKey);
参数就是要关闭的那个句柄

读取函数

函数原型：
LONG RegQueryValueEx(
HKEY hKey,
LPCTSTR lpValueName,
LPDWORD lpReserved,
LPDWORD lpType,
LPBYTE lpData,
LPDWORD lpcbData
);

HKEY hKey:
子键句柄
LPCTSTR lpValueName:
键值名称
LPDWORD lpReserved:
为0
LPDWORD lpType:
返回键值的数据类型
LPBYTE lpData:
返回键值的数据
LPDWORD lpcbData：
缓冲区的大小

函数调用成功，返回ERROR_SUCCESS

写入函数

当然了，前面说的RegCreateKey()就已经可以做到写入了
但是写入函数还可以用RegSetValueEx
函数原型：
LONG RegSetValueEx(
HKEY hKey,
LPCTSTR lpValueName,
DWORD Reserved,
DWORD dwType,
const BYTE* lpData,
DWORD cdData
);
HKEY hKey:
键值所在子键的句柄
LPCTSTR lpValueName:
要设置的键值名称
DWORD Reserved:
为0
DWORD dwType:
设置键值的数据类型
const BYTE* lpData:
设置键值的数据缓冲区
DWORD cdData:
设置键值的数据缓冲区大小

删除键值函数

函数原型：
LONG RegDeleteValue(
HKEY hKey,
LPCTSTR lpValueName
);
HKEY hKey:
子键的句柄
LPCTSTR lpValueName:
删除键值的名称
